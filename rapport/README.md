# Rapport LaTeX — StoryTranslator (INF39615)

## Fichiers

| Fichier | Rôle |
|---------|------|
| `main.tex` | Document principal (contenu complet) |
| `includes.tex` | Packages et macros |
| `titlepage.tex` | Page de titre |

## Overleaf

1. Créer un projet Blank
2. Uploader `main.tex`, `includes.tex`, `titlepage.tex`
3. (Optionnel) Ajouter le logo `uqar.png`
4. Compiler en **PdfLaTeX**

## Local

```bash
cd rapport
pdflatex main.tex
pdflatex main.tex
```
